### 기본 구현 사항

- [x] vanilla js 기반의 carousel 구현
- [x] 현재 슬라이드의 index 를 알려주는 인디케이터
  - [x] 형태는 자유
  - [x] 가능하면 동그란 모양의 바닥에 배치
  - [x] 인디케이터를 누르면 해당 index 로 바로 이동
- [x] 앞, 뒤의 슬라이드로 넘어갈 수 있는 버튼 필요
  - [x] 화면에 좌, 우에 배치하고자 함
